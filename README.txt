SPERIN INTERIORS WEBSITE

How to run locally:
1. Install Node.js
2. Open a terminal in this folder
3. Run: npm install
4. Run: npm run dev
5. Open: http://localhost:3000

How to put it live on Vercel:
1. Create a Vercel account
2. Click Add New Project
3. Upload this folder to GitHub or import it directly if available
4. Click Deploy
5. In Vercel project settings, add your domain: sperininteriors.uk
6. Update the DNS records at your domain provider to the values Vercel gives you
